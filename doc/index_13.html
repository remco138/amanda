random


random :: num -> num

random n = a random number r with 0 <= r < n (n integer, n > 0)

random should be used with care, because it is not referentially transparent.

Example:
[r | k <- [1..100]; r := random 10] ~= [r | r := random 10; k <- [1..100]]
